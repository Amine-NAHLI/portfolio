## Purpose

Toolkit for testing local web applications with Python Playwright — server lifecycle management, UI verification, screenshots, browser logs, and reconnaissance-then-action automation patterns.

## Why selected

- **Fills the testing category** — official, well-documented web app testing skill with bundled helper scripts
- **Practical automation** — includes `with_server.py` for managing dev servers during tests
- **Agent-friendly design** — scripts intended to run as black-box helpers without polluting context
- **General-purpose** — works with any local web stack (static HTML or dynamic SPAs)
- **Trusted source** — Anthropic official skill with strong ecosystem presence on [skills.sh](https://skills.sh/anthropics/skills/webapp-testing)

## Purpose

Teaches coding agents Vercel Engineering's React and Next.js performance optimization guidelines. The skill covers 70 rules across 8 categories — from eliminating async waterfalls and bundle bloat to server-side caching, re-render optimization, and rendering performance — with incorrect/correct code examples for each rule.

Use it when writing, reviewing, or refactoring React components, Next.js pages, data fetching, or bundle optimization work.

## Why it was selected

- **Authoritative source** — maintained by Vercel Engineering, backed by 10+ years of React/Next.js production experience
- **Impact-prioritized** — rules ranked CRITICAL → LOW so agents fix high-impact issues first

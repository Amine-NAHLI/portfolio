# Amine Nahli — Portfolio

> Security Engineer × Full-Stack Builder

Personal portfolio website built with Next.js 16, TypeScript, Tailwind CSS v4, Three.js, and Framer Motion.

**All project data is fetched dynamically from the GitHub API** — when you create a new repo, it appears automatically on the site.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **3D:** Three.js + React Three Fiber
- **Animations:** Framer Motion
- **Scroll:** Lenis smooth scroll
- **Data:** GitHub REST API (dynamic, ISR revalidation every hour)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Deploy

Deploy on [Vercel](https://vercel.com) for automatic ISR support.
